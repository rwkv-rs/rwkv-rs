# 简介 / Introduction

## 什么是 rwkv (读作 RWaKuV)
广义的 rwkv 是围绕线性复杂度的循环神经网络(LinearRNN)建立的一整套精心设计的深度学习生态体系(每个组件缺一不可), 而狭义的 rwkv 则是这个星球上首个具有 GPT 级大语言模型(LLM)性能且可并行化训练的 RNN LLM.

rwkv 结合了 RNN 和 Transformer 的最佳特性: **出色的性能, 恒定的显存占用, 恒定的推理速度, 理论无限长度的上下文和免费的句嵌入**, 而且完全不含自注意力机制. 随着 rwkv 生态的发展, 线性复杂度 RNN 的理论缺陷也将在 rwkv-v8 版本中被弥补, 是 Transformer 生态的强力竞争者.

rwkv 项目最初由彭博(Bo Peng, BlinkDL)提出, 随着项目被外界关注, 逐渐发展成一个开源社区, 而 **rwkv-rs** 将会是其中涵盖面最为广泛, 工程化程度最高的开源项目.

2023 年 9 月 20 日, rwkv 开源项目正式加入 Linux 开源基金会. 如今的 rwkv 团队是一个开源的非盈利组织, 隶属于 Linux 基金会, 部分算力曾由赞助商支持. 

## 什么是 rwkv-rs
**rwkv-rs** 是一个用 Rust 编写的 rwkv 的 **任意** 模型训练/推理/评估/研究/Agent框架, 基于[Burn](https://github.com/tracel-ai/burn) 深度学习框架构建. 它提供从数据预处理、模型训练到生产级推理服务等的完整工具链. 你可以使用 **rwkv-rs** 设计 **任意** 自己的 大语言模型 / 视觉模型 / 时间序列模型... 或 **任意** 你感兴趣的下游任务.

## 为什么用 Rust
- **工程化**: 现有 rwkv 生态下, 几乎大部分项目都为研究型项目, 真正代码 **易于理解** 的, 项目 **易于部署** 的, 且能 **用于生产环境** 的项目, 极少极少. 我们的核心目标便在于改变这一现象, 从而为 rwkv 生态的发展铺平道路.
- **跨平台**: 统一后端抽象, 同一套代码支持 CUDA / ROCm / Metal / Vulkan / WGPU.
- **性能**: 零成本抽象, 无 GC 停顿, 多线程 & 多进程. 
- **安全**: 内存安全, 线程安全, 无数据竞争, 适合多 GPU 并发场景.

## 本书结构

本书按照"先跑起来 → 了解用途 → 最佳实践 → 深入原理"的顺序组织：

1. **快速上手** — 5 分钟内跑起训练 / 推理 / 评估
2. **数据处理** — 数据去重 / 去乱码 / 数据合成 / 分词器
3. **模型设计** — WeightsPrepare / WKV Kernel / GatedReadout / ChannelMix / PostNorm / ValueResidual
4. **训练教程** — 预训练 / 后训练 / 微调 / 风格对齐
5. **推理教程** — OpenAI API & 进程间通信 / 批调度 / 采样参数
6. **评估教程** — 传统 Benchmark / UncheatableEval / 刷榜

## 面向用户
- **研究者**: 希望基于 rwkv 进行改进以追求某领域 sota 的用户
- **企业用户**: 希望训练或微调, 并进行私有化部署高性能 LLM 的用户
- **LLM 发烧友**: 希望创造一个独属于自己, 有着独特风格 LLM 的用户
- **Rust 爱好者**：希望了解如何在 Rust 中构建深度学习程序 的用户

不需要深入了解 Rust 即可开始使用, 但建议阅读 [Rust Book](https://doc.rust-lang.org/book/) 前几章.
